Step 2.1: Inset Math Implementation

Padding (Inset)

Avoid the "Edge-Cut" problem by moving the first and last labels away from the chart boundaries.

Ratio Math

Calculate the positionRatio between inset and 1-inset.

Interpolation

"Linear interpolation across indices ensures labels are perfectly centered relative to data."


const getInsetEvenlySpacedIndexes = (indexes, labelCount, insetRatio = 0.05) => {
  const lastIndex = indexes.length - 1;

  return Array.from({ length: labelCount }, (_, pointIndex) => {
    // Math: Linearly interpolate between [inset] and [1 - inset]
    const positionRatio =
      insetRatio + ((1 - insetRatio * 2) * pointIndex) / (labelCount - 1);

    const targetIndex = Math.round(lastIndex * positionRatio);
    return indexes[targetIndex];
  });
};
